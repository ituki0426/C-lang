# ```&``` ```|``` ```^``` ```~```

それぞれ、ANDOR XOR NOT(1の補数)


NOTはビット反転を行う

# ```>>``` ```<<```


2進数の数を指定したビット数だけ左右にずらす


# 例


➀

```c
#include <stdio.h>
int main(void){
int i,k;
scanf("%d",&k);
for(i=31;i>=0;--i){
printf("%d", (k >> i) & 1);
}
printf("\n");
return 0;
}
```
